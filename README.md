# xScript
A powerful Plug.DJ script.
To load the script in a Plug community, drag this bookmarklet to your bookmarks bar, and run it on the Plug page.
[xScript Bookmarklet](javascript:$.getScript('https://cdn.rawgit.com/phantamanta44/xScript/master/xscript.js'))

Alternatively, you can use the following bookmarklet to run the in-dev version, which contains all the freshest features (but isn't guaranteed to run smoothly, or even at all).
[In-dev Bookmarklet](javascript:$.getScript('https://dl.dropboxusercontent.com/u/75796840/xScript/xscript.js'))

jQuery script loading functions:
```
// GitHub version
$.getScript('https://cdn.rawgit.com/phantamanta44/xScript/master/xscript.js');

// Dropbox in-dev version
$.getScript('https://dl.dropboxusercontent.com/u/75796840/xScript/xscript.js');
```
